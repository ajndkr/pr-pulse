# pr-weekly

generate weekly reports of all code changes in a github repository.

## usage

run the following to view the list of available commands.

```shell
make run
```

## getting started

### pre-requisites

- [uv](https://docs.astral.sh/uv/#getting-started)

### init project

```shell
make init
```

to run pre-commit checks, run:

```shell
make ci
```
